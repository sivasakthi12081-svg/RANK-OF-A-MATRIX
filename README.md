# RANK-OF-A-MATRIX
## Aim:
To write a python program to find the rank of a matrix
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:
### Step 1: Start the program
Import the required library (numpy) to perform matrix operations.
### Step 2: Initialize the matrix
Create a matrix A using numpy.array() with the given elements.
### Step 3: Compute the rank
Use the function np.linalg.matrix_rank(A) to calculate the rank of matrix A.
### Step 4:Display the result
Print the computed rank on the screen using print().
## Program:
```
#Program to find the rank of a matrix.
#Developed by: Sivasakthi S
#RegisterNumber:25017123
import numpy as np
A=np.array([[5,-3,-10],[2,2,-3],[-3,-1,5]])
rank=np.linalg.matrix_rank(A)
print(rank)
```
